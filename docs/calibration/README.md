# Calibration documentation

This directory is the repository-level entry point for SlopBrick calibration
work. The package documentation remains authoritative because the calibration
runner, schemas, and evidence are owned by `packages/slopbrick`.

## Authoritative sources

- [Continuation plan](../../packages/slopbrick/docs/calibration/v0.45.0-continuation-plan.md) — scope, gate requirements, release stops, and the current implementation sequence.
- [Execution evidence](../../packages/slopbrick/docs/calibration/v0.45.0-execution-evidence.md) — evidence already collected and work that is still open.
- [Handoff](../../packages/slopbrick/docs/calibration/v0.45.0-handoff.md) — historical context; its supersession notice and the continuation plan take precedence.
- [Package calibration README](../../packages/slopbrick/docs/calibration/README.md) — legacy workflow context. Its historical absolute-path commands are diagnostic-only, not release evidence.

Use [v0.45-continuation.md](./v0.45-continuation.md) for the current
Gate 3 → Gate 4 dependency sequence. Do not duplicate, move, or treat these
repository-level pointers as calibration evidence.
